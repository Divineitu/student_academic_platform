# ALU Academic Assistant App

Mobile app for ALU students to track assignments, schedules, and attendance.

## Project Structure
- `lib/screens/` – App screens (Dashboard, Assignments, Schedule)
- `lib/models/` – Data models
- `lib/constants/` – Colors and constants

## Getting Started
1. Clone the repository
2. Run `flutter pub get`
3. Run `flutter run`
